Exemplo de um elemento HTML

Para criar um parágrafo em HTML, você deve usar a tag <p>. Tudo o que estiver dentro dessa tag será exibido como um bloco de texto separado, com espaçamento automático acima e abaixo. Por exemplo, o código abaixo cria um parágrafo com o texto "Olá, Mundo!": 

    
<p>Olá, Mundo!</p>
